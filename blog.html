Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model represents the structure of an HTML document as a tree of object. It is the programming interface of a web documents.There are many nodes like elements, attribute, text etc. We can easily interact dynamically and make changes in the websites.

How do you select an element from the DOM?

We can easily select an element from the DOM. At first we have to give an id to that particular element. Then in our js file we have to create a variable to store the selected element. Then we have to write document.getElementById("id"). So the element will store in the variable. We can also use querySelector(). In this case we have to add the css selector sign before the id or class. And there are also many way to select an element.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a event listener is attached to a parent element. So that it will handle events triggered by its child element. It manage the child element using event bubbling.

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes using setAttribute() and getAttribute() methods. For styling we have to use style property. Example: element.style.backgroundColor = "red". It will set the background color to red.
